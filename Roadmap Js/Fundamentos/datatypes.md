# Tipos de dados

No Js há 8 tipos de dados básicos, onde vamos trabalhar com eles uma grande parte de nossa jornada. Vale ressaltar que como Js não é fortemente tipada, o tipo de dado de uma váriavel pode mudar com o passar do tempo, um exemplo disso é que a váriavel pode tanto armazenar originalmente um valor númerico para em seguida armazenar uma string, para isso ocorrer só é preciso declarar o valor da maniera correta.

- **Number** - Represanta todos os números, dede positivos negativos e quebrados com virgula (float).
- **BigInt** - Usuado númericamente para valores ridiculamente grandes, quando utilizado é preciso colocar um {n} no final do número Exp: ..123n; .
- **String** - Usado para guardar letras, podendo ser nulo, uma ou várias letras. Sempre que for utilizado para guardar valores utilizar aspas simples ou duplas, já as crases são usadas para fazer interpolações como no c#.
- **Boolean** - Armazena valores booleanos como verdadeiro ou falso.
- **Null** - Declara uma variavel com um valor vazio, literalmente nada.
- **Symbol** - Cria um valor único, quando uma váriavel é um symbol ela é única e geralmente pode ficar escondida em metodos para não dar conflito com nomes de outras variáveis, pois mesmo que tenha o mesmo valor um symbol nunca é igual a outro, ele é simplesmente único.

Esses são os tipos de variaveis primitivos, para o futuro podemos ver objects, onde podemos guardar divresos tipos de dados em um lugar só, assim sendo mais complexo e não entrando inicialmente nessa lista de tipos de dados primitivos.
