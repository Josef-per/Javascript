# Introdução ao JavaScript

Nessa parte vamos ter todos os assuntos retratados no dia 1, sendo todo um contexto do Js, sendo pra que ele serve, versões, história e outros.

## O que é JavaScript

Js ou JavaScript, é uma linguagem de programação que foi desenvolvida inicialmente para dar vida a web pages, adicionando movimentos ações triggers e outros, contudo hoje com o avanço da linguagem é indiscutivel que não para por ai, sendo utilizada em servidores, web, mobile e até aplicações desktop.

Muitos navegadores tem conversores de código Js dentro deles sendo um dos mais comuns o V8, utilizado no chrome opera e o edge, contudo há diversos outros motores que fazem o Js ser viável para web, contudo esses são os principais. Sendo de extrema importância para a tecnologia sem eles a linguagem não seria o que é hoje

## História do Js

O obejtivo não é estudar a história da linguagem, sendo assim vou abordar assuntos bem rasos.

Sendo criada por Brendan Eich, tendo seu primeiro anúncio em 1995, tendo uma história longa de trocas de nomes ao longo do tempo dede Mocha a LiveScript, contudo como uma tentetiva de monetivar, surfando no hipe de Java, Brendan coloca o nome de JavaScript, mesmo que Java e JavaScript não sejam nehum um pouco semelhantes.

## Versões do Js

O JavaScript trabalha exclusivamente no EcmaScript, na qual as atualizaçõs nessa parte de sistema influencia diretamente a linguagem, pois o EcmaScript que define a syntaxe, semântica, ou seja ela pode trocar tudo que está dentro e funcionando na linguagem.

O EcmaScript já teve diversass verções, onde todas são chamadas de ES(x), onde o x é o número da versão, hoje em dia estamos na versão ES2025, desde 2016, pararam de numerar do de forma mais normal sendo do 1, 2 ... e assim pordiante, começando a colocar o ano na frente da versão lançada.

Temos versões importantes que deu uma evolucionada na linguagem, sendo a primeira a ES5, conhechida como Modern JavaScript, ou javascript moderno, onde diversas funções muito utilizadas hoje foram incrementadas, desde listas, arrays, fazendo a sua utlização ser bem mais fácil.

## Como rodar Js

Para que o Js possa rodar, fazemos um arquivo com a terminalção .js, para que ele seja visto e interpretado como um arquivo JavaScript, esse arquivo pode ser tanto executado tendo sua resolução mostrada no terminal, ou podemos acionar em arquivos html, para termos ele dentro de um site simples como um exemplo.

Para adiconar em sites como um html, precisamos somente fazer uma conexão com a tag <script scr="">
