# Estudos de JavaScript

Este repositório documenta meu aprendizado em JavaScript,
seguindo um roadmap estruturado, com exemplos, exercícios
e projetos práticos.

## Estrutura
- Roadmap Js: fundamentos, exemplos e exercícios
- Projetos: aplicações práticas
- Diário Técnico: anotações e aprendizados
- Estudos Semanais: planejamento e progresso

## Objetivo
Construir uma base sólida em JavaScript para evoluir
para TypeScript, React e desenvolvimento backend.
